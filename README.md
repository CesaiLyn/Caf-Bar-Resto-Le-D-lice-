# 🍽️ Le Délice

Bienvenue sur le dépôt officiel du site web du restaurant **Le Délice**.

## 📖 À propos

Le Délice est un restaurant qui propose des plats savoureux préparés avec des ingrédients frais et de qualité, dans une ambiance chaleureuse et conviviale.

Ce site a été créé pour présenter le restaurant, son menu et ses informations de contact.

## ✨ Fonctionnalités

- Page d'accueil avec une image de présentation
- Section « À propos »
- Présentation des spécialités du restaurant
- Informations de contact
- Design adapté aux ordinateurs et aux téléphones

## 🛠️ Technologies utilisées

- HTML5
- CSS3
